# FitSyn - Your Premium Fitness Hub

FitSyn is your AI-powered fitness companion designed to help you train smarter, get stronger, and transform your life.

## Features

- **AI-Powered Workouts**: Personalized training programs tailored to your goals
- **Real-time Tracking**: Monitor your progress with detailed analytics
- **Yoga & Meditation**: Mind-body balance for complete wellness
- **Nutrition Guidance**: Smart meal planning and calorie tracking
- **Interactive Games**: Fun fitness challenges to keep you motivated

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/fitsyn.git

# Navigate to the project directory
cd fitsyn

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Framer Motion
- Radix UI

## Development

```bash
# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm run test
```
